Github Repository  I am sure you know this as you are already viewing this page, but I created a website to host the entirety of my portfolio. The whole site was created just using raw HTML5 and CSS3 of course. I wanted it to look simple/minimalistic while still looking visually appealing. I purchased a .io domain as it shares its two letters with the ending of the word portfolio, and my last name.
  I am locally hosting this site on my own server, running Apache on Ubuntu. I used a script that automatically runs every few minutes to update my DNS records to match my dynamic WAN IP address. This way I do not need a static IP address.
